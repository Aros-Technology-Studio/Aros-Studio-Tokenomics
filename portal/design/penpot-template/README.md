| `04-explore.svg` | Public search |
| `05-login.svg` | Login |

## Import into Penpot

1. Open project → file  
2. **Menu → File → Import** (or drag SVG onto canvas)  
3. Import each SVG as its own board  
4. Ungroup if needed; replace labels with your text/colors  

Path on disk:

`portal/design/penpot-template/`
