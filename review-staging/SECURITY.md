# Security Policy

## Reporting a Vulnerability
If you discover a security issue in this project, please email our security team at [security@aros.network](mailto:security@aros.network). We aim to acknowledge your report within 48 hours and provide an initial response, including next steps, within 5 business days.

## Supported Versions
Only the latest minor release is supported with security updates.

| Version | Supported |
| ------- | --------- |
| 0.1.x   | ✅ |
| < 0.1.0 | ❌ |

## Requesting CVE IDs
We will work with reporters to request CVE identifiers for confirmed vulnerabilities. After triage and a fix or mitigation is in place, our team will coordinate CVE publication through the appropriate CNA.
