# API — AllSeeingEyeService

| Method | Description |
|--------|-------------|
| `observe` | Record an observation event |
| `notify` | Observe with alert intent |
| `subscribe` | Register listener |
| `history` | Audit log of All-Seeing Eye events |
| `veto` | Always throws — All-Seeing Eye has no veto |
